# Enterprise-Campus-Network

This project simulates a small enterprise campus network using Cisco IOSv and IOSvL2 devices in PNETLab.

The network is segmented using VLANs for different departments, uses multilayer switching for inter-VLAN routing, OSPF for dynamic routing, and ACLs for basic access control.

## Objectives

- Implement VLAN segmentation
- Configure trunk links
- Perform inter-VLAN routing
- Deploy OSPF dynamic routing
- Apply ACL security policies
- Verify end-to-end connectivity
